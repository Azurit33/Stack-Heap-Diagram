# Stack-Heap Diagram

## Description
Stack and heap diagram for a Java programme demonstrating:
- JVM memory allocation
- Local variables in the stack
- Objects in the heap
- Relationships between them

## Files:
- `Main.java` - program source code
- `stack-heap-diagram.png` - visual diagram

## How the assignment was completed:
1. Analysis of the program code
2. Construction of a step-by-step diagram
3. Verification of all relationships
